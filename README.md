# Tree-Coverage-in-Urban-Environments-for-Community-Health-and-Energy-Saving
Sarah Brown
University of Calgary - Department of Geography
sarahkbrown0gmail.com

The purpose of this project was to analyze tree coverage density impacts on surface temperature around calgary and how the presence of trees may be unequal across the city due to income disparities between communities.

Land_Surface_Temperature_Calculations contains all ArcGIS layers used to calculate land surface temperature in the city of Calgary from 2020/07/26 and 2022/07/24.
Tables_Created contains attribute table of Calgary community data used to create map of income and map of tree are coverage by community, and excel file of correlation between tree coverage and income.
Tree Canopy files contain geo_export files of tree coverage in calgary
